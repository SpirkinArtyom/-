# Порядок разработки и развертывания контейнеров JPA и EJB на сервере приложений

## 1. Создание базы данных
1. Создайте базу данных и таблицы, используя SQL-запросы.
2. Заполните таблицы минимальными данными.

## 2. Настройка JDBC-соединения и пула соединений
1. Настройте сервер приложений (например, Tomcat) для использования пула соединений, добавив ресурс в файл `context.xml`.
2. Настройте веб-приложение для использования этого ресурса, добавив описание ресурса в файл `web.xml`.

## 3. Разработка JPA-контейнера
1. Создайте классы сущностей для каждой таблицы базы данных, используя аннотации JPA.
2. Настройте файл `persistence.xml` для указания источника данных и классов сущностей.

## 4. Разработка сессионного компонента и EJB-контейнера
1. Создайте сессионный компонент (EJB) для взаимодействия с сущностями.
2. Создайте сервлет для обработки запросов и взаимодействия с EJB.

## 5. Развертывание и тестирование
1. Экспортируйте проект в WAR файл.
2. Разверните WAR файл на сервере приложений.
3. Проверьте функциональность приложения, убедившись, что все компоненты работают корректно.
